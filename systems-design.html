SYSTEMS DESIGN

General Interaction System

The interaction system is an actor component that can be added to the player to make them interact with certain blueprints. It uses collisions to detect when the player is within the range of an interactable object and displays the appropriate UI element. The system supports different types of interactions, such as single press, hold for any duration (instance editable), and tap repeatedly any number of times (instance editable).

X-Ray Vision

X-Ray Vision is a post-processing effect that allows the player to see through walls and other objects to locate enemies and items of interest. The effect is achieved using a custom depth pass and a post-process material that highlights objects based on their distance from the player. The system is designed to be modular and can be easily integrated into any Unreal Engine 5 project.

It can be particularly useful in stealth missions or when navigating complex environments where visibility is limited, as includes different colors for different object types, such as enemies and items, to enhance situational awareness.

Enemy AI

(WIP) This is a comprehensive series of youtube videos by Ali Elzoheiry on different types of enemy AI, including melee, ranged, and magic enemies. The AI uses behavior trees and blackboards to manage their states and actions, allowing for complex behaviors such as patrolling, chasing, and attacking the player.

This was part of a learning exercise, during my time as a systems designer, to understand AI programming in Unreal Engine 5. The system is designed to be modular and can be easily integrated into any Unreal Engine 5 project.

Parkour System

(WIP)The parkour system is a comprehensive movement system that includes various mechanics such as sprinting, mantling, sliding, rolling, grappling hook, and different types of jumps (vault, double jump, triple jump, spin jump, and long jump).

This will be an important part of my Daredevil fan game as Daredevil focuses almost completely on combat and parkour. It is an ongoing project, and I'm learning from a long list of youtube videos to implement these mechanics using Blueprints in Unreal Engine 5.
